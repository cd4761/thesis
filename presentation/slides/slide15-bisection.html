5. 오프체인 이등분 탐색

핵심 기여

기존 방식의 문제

온체인 이등분 탐색

→ 각 라운드마다 온체인 거래

→ 높은 가스 비용 + 긴 해결 시간

제안 방식

ZK 상태 채널 기반 오프체인 이등분

→ 온체인 비용 없이 빠른 탐색

→ ZK 서명으로 부인 방지

1백만 명령어 탐색

~20 라운드

탐색 복잡도

O(log N)

소요 시간

수 분 내 완료

프로토콜 흐름

1

분쟁 발생 시 상태 채널 개설

2

오프체인에서 이등분 탐색 수행

3

각 상태 전이에 ZK 서명 첨부

4

분쟁 지점 (단일 명령어) 식별

5

결과를 온체인에 제출
