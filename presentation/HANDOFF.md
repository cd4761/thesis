## 실패/미완료 작업

### 1. 썸네일 그리드 생성 실패 ❌
- **원인**: LibreOffice가 설치되어 있지 않음 (Windows 환경)
- **시도한 명령**: `python scripts/thumbnail.py dissertation-defense.pptx thumbnails --cols 4`
- **에러**: `[WinError 2] 시스템이 지정된 파일을 찾을 수 없습니다`
- **해결 필요**: LibreOffice 설치 또는 다른 PDF 변환 도구 사용

### 2. html2pptx 스킬 사용 실패 ❌
- **원인**: Playwright 글로벌 설치 경로 문제
- **우회 방법**: `create-pptx-direct.js`로 PptxGenJS 직접 사용
